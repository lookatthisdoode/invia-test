# Hotel Rooms Availability

This project is a React-based frontend for displaying hotel room availability and prices. Users can sort and paginate through rooms, check availability, and book rooms.

## Table of Contents
- [Features](#features)
- [Getting Started](#getting-started)
- [Running the Project](#running-the-project)
- [Contributing](#contributing)
- [License](#license)

## Features
- Display a list of hotel rooms with pagination.
- Sort rooms by different criteria.
- Check availability of rooms with a simulated loading state.
- Book rooms with updated pricing.
- Maintain initial sorting after price updates.

## Getting Started

### Prerequisites
- Node.js and npm installed on your machine.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/lookatthisdoode/invia-test.git
   cd invia-test
   ```
2. Install dependencies:
    ```bash
   npm install
   ```

### Running the Project
To start the development server, run:
   ```bash
   npm run dev
   ```
The application will be available at http://localhost:3000.
